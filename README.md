# 01-fundamentos-react-js

A **social feed** web application built during the [Rocketseat Ignite](https://www.rocketseat.com.br/ignite) bootcamp (Module 1 — JavaScript track).

## Objective

The objective of this project is to learn the fundamentals of React — JSX, components, props, state, and event handling — by building a social network-style feed where users can browse posts and interact with comments.

## Features

- Feed of blog-style posts with author info and formatted dates
- Sidebar displaying user profile
- Add and delete comments on posts
- Applaud (like) individual comments

## Tech Stack

- React 18
- JavaScript (JSX)
- Vite
- CSS Modules
- date-fns (date formatting)

## Getting Started

```bash
yarn install
yarn dev
```
